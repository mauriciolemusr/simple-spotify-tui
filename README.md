# spotify-tui

A terminal UI for controlling Spotify.

![screenshot](screenshot.png)

## Install

```
pipx install git+https://github.com/mauriciolemusr/simple-spotify-tui.git
```

Or with pip:

```
pip install git+https://github.com/mauriciolemusr/simple-spotify-tui.git
```

## Usage

```
spotify-tui
```

On first run you'll be walked through creating a Spotify app and authorizing access.

## Keybindings

| Key       | Action      |
|-----------|-------------|
| Space     | Play/Pause  |
| n         | Next track  |
| p         | Prev track  |
| + / -     | Volume      |
| l         | Like/unlike |
| s         | Shuffle     |
| r         | Repeat      |
| /         | Search      |
| t         | Cycle theme |
| Tab       | Switch focus|
| Esc       | Back        |
| q         | Quit        |

## Prerequisites

- Spotify Premium account
- Python 3.11+
- Linux or macOS
- A Spotify Connect device running somewhere (desktop app, phone, web player, spotifyd, etc.)
